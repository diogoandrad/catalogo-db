# Docker MicrosoftSQL database
Container MicrosoftSQL

# Run

Clone the project:
```bash
git clone https://github.com/diogoandrad/docker-microsoftsql.git
```

Enter the project folder:
```bash
cd docker-microsoftsql
```

Build the image:
```bash
docker compose build
```

Run the image:
```bash
docker compose up
```
